# Axiomtek AMI Megarac SP MOD N1

Мод BMC на базе стокового Axiomtek AMI Megarac SP 2.02.76714.

```ini
FW_VERSION=2.02.76714
FW_DATE=Aug 8 2022
FW_BUILDTIME=MOD N1 VER01
FW_DESC=RR10 AST2500 MOD N1 VER 01
FW_PRODUCTID=1
FW_RELEASEID=RR9
FW_CODEBASEVERSION=3.X
```

## Changes
- Это чистый BMC без хвостов от любых запусков
- MAC изменён на `de:ad:be:ee:ee:ef`
- Консоль изменена `ttyS4` => `ttyS1` (это колодка `IPMICOM1` на доске)
- Встроен хак [с рутовым шеллом на ssh](../Megarac_SP_2.02.76714/index.mdx#root-shell)

## Serial console
```yaml
Port: IPMICOM1
Baudrate: 38400
```
