# Depo TianoCore EDK2 MOD L1

Mod на базе сборки EDK2 BIOS от Depo.

```yaml
vendor: TianoCore EDK II Project
```

## Changes

- Добавлены микрокоды 606A4 и 606A5
- Обновлен микрокод 606A6 до последней версии

:::note
606A5 и 606A6 проверены запуском с QVM7 и QWAT
:::
