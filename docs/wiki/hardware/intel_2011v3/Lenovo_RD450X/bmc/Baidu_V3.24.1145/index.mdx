## Manual fan control

–î–ª—è –≤–µ—Ä—Ç—É—à–µ–∫ —Å–º–æ—Ç—Ä–∏—Ç–µ –±–ª–æ–∫ **3.3 Set Fan Configuration Command** in [Lenovo BMC Command List V0.1.1](@storageBaseUrl@/Lenovo-RD450x/docs/Lenovo-BMC-Command-List_V0.1.1.pdf)

–û–±—â–∏–π –∫–æ–Ω—Ü–µ–ø—Ç —Ç–∞–∫–æ–π:

```csharp title="ManualFanControl.cs"
public enum LenovoIpmiFanType
{
    ALL = 0,
    SYS_FAN1 = 1,
    SYS_FAN2 = 2,
    SYS_FAN3 = 3,
    SYS_FAN4 = 4,
    CPU_FAN1 = 5,
    CPU_FAN2 = 6,
}

public async Task SetFanSpeedAsync(LenovoIpmiFanType fan, int percentage)
{
    if (percentage > 100)
        percentage = 100;

    if (percentage < 0)
        percentage = 100;

    new ProcessStartInfo()
    {
        FileName = _options.IpmiToolFile ?? "ipmitool",
        Arguments = $"{_options.AccessArgs} raw 0x2e 0x30 00 {(int)fan} {percentage}",
        RedirectStandardError = true,
        RedirectStandardOutput = true,
        RedirectStandardInput = false,
    }
}
```
